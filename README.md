# «Теледок»
## Описание
Веб-приложение для добавления и просмотра клиентов (редактирование по желанию). Клиентами выступают юридические лица
(ЮЛ) и индивидуальные предприниматели (ИП). ЮЛ могут иметь некоторое количество учредителей.
